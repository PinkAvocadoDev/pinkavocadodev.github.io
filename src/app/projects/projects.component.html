Posso Andare Al Mare?

"Can you go to the beach today?" Here's your answer.

Posso Andare Al Mare is a web app that gives a simple, direct answer — Yes or No — to the question: "Is today a good day to go to the beach?" It covers nearly all Italian beaches and offers forecasts up to 7 days in advance, all in a sleek modern GUI.

The app fetches weather data from the OpenMeteo.com API and processes it to provide an intuitive, easy-to-understand result. For those who want more detail, a popup shows wind speed, average temperature, sun intensity, cloud coverage, and more.

Built using PHP, HTML, CSS, and JavaScript, the app is live at possoandarealmare.altervista.org.

Bonus: It meets modern web standards and can be installed as a Progressive Web App (PWA) via Chromium or Safari.

TF2 Web Remote

A Web-Based Control Panel for Team Fortress 2 Servers.

TF2 Web Remote is a project I developed in collaboration with two friends. It provides a sleek and user-friendly web interface for managing personal Team Fortress 2 servers.

With this control panel, users can easily:

  • Select the game mode and map stage;
  • Adjust the number of bots;
  • Monitor and manage server settings in real time.

The backend is powered by a Python server that communicates directly with Valve’s native server API, sending commands to the remote machine. The setup is fully customizable through a config.yml file, where you can define parameters like IP address, ports, and other admin preferences.

VenPatch

A GTK4 and Libadwaita app for Vencord patching.

I love Vencord—the themes, icons, and added utilities are fantastic! But there’s a catch: Discord updates frequently, and every update tends to break Vencord, requiring it to be re-patched.

If you're on Windows (ew), this isn't a big deal—the Vencord team provides a handy re-patching utility with a simple click-to-run interface. On Linux, though, you're left re-running a curl command to download and execute the Go-based installer script. Tedious.

That’s why I made VenPatch—a GTK4 and Libadwaita app that wraps the official Vencord installer in a clean, modern GUI. With VenPatch, you can Install, Repair, or Uninstall Vencord with just a few clicks.

Bonus: VenPatch is fully compliant with the GNOME Human Interface Guidelines (aka the GNOME Design Bible).
